Scrapy基本使用

环境准备

pip install scrapy

项目操作

在当前目录创建scrapy项目
scrapy startproject [project_name]
scrapy startproject test_scrapy

进入项目目录，生成爬虫文件
scrapy genspider [name] [url]
scrapy genspider first www.xxx.com

运行项目
scrapy crawl [name]
scrapy crawl first
