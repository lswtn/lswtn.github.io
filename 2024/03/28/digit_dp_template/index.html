数位dp模板

模板

应用：求某个区间[l, r]中，满足某个条件的数的个数
思路：

  1. 函数f(x)求[0, x]合法的数的个数，答案为f(r) - f(l-1)
  2. 从高位向低位枚举，每一位t可以划分为两个分支，左分支为0,1,…,t-1，右分支为t
  3. 预处理左分支的所有合法个数，递归处理右分支
  4. 不重不漏动态规划[0, x]所有数
