1 ~ n所有数的约数个数和 ≈ nlogn
int范围内约数最多的一个数的约数个数 ≈ 1600
如果 N = p1^c1^ * p2^c2^ * … *pk^ck^ (算术基本定理)
约数个数： (c1 + 1) * (c2 + 1) * … * (ck + 1)
约数之和： (p1^0^ + p1^1^ + … + p1^c1^) * … * (pk^0^ + pk^1^ + … + pk^ck^)
